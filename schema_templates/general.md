# Wiki Schema

## Purpose
This wiki tracks a personal knowledge base curated from documents dropped into the inbox.

## Folder Structure
- **Sources/**: one summary note per ingested source.
- **Entities/**: one note per person, company, or product.
- **Concepts/**: one note per topic or technique.
- **Synthesis/**: notes that connect multiple sources.

## Note Format
- Summary
- Key Points
- Source
- Related Notes

## Cross-References
Use `See: <Page Title>` inline and include a final `## Related Notes` section.

## Index Format
The `_Index` note should list wiki pages by subfolder with a one-line description for each.
